# 常見問題 Q&A

> 開始上課、或卡關時，先翻這裡。大多數狀況這份都能幫你解掉。

## 🧰 課程與工具

### Q：我完全不會寫程式，學得會嗎？

可以。整套課程的設計就是**不用懂程式**——你只要會「打字跟 Nora 說話」「複製貼上」就行。遇到看不懂的指令，直接把它貼給你的 Claude，問「這是什麼、我該怎麼做」，它會用白話帶你。

### Q：一定要付費的 Claude Pro 嗎？免費版能做什麼？

不一定。**免費版（claude.ai 網頁版）該有的「腦」一個都不少**——五位成員的問答、規劃、寫作、策略能力完整，差別只在「手腳」（自動發文、自動拉數據這些自動化）。

- **先用免費版**把流程跑順、把內容做出來，手動複製貼上發布，完全沒問題。
- 需要自動化（排程發文、自動拉廣告數據）再升級 **Claude Pro（約 US$20／月）**。

免費版用法見 [免費版專區](free-version/)。

### Q：Mac 和 Windows 都能用嗎？

都可以。Claude Code 已原生支援 Windows，**不需要 WSL**。安裝指令我們也都附了 Mac 與 Windows 兩種版本，見 [INSTALL.md](INSTALL.md)。

## 🛠️ 安裝與更新

### Q：裝完輸入 `/hello-nora` 卻說「找不到指令」？

通常是這幾個原因，照順序檢查：

1. **沒複製到位**：確認 `~/.claude/skills/hello-nora/`（Windows：`C:\Users\你\.claude\skills\hello-nora\`）裡真的有檔案。沒有就回 [INSTALL.md](INSTALL.md) 的 Step 1 重做一次。
2. **沒「完全」重開**：安裝後一定要**完全關閉 Claude Code 再重新打開**——是「結束 Claude」（Mac ⌘Q／Windows 工具列圖示右鍵→結束），不是按紅色叉叉縮小。指令才會被讀到。
3. **（免費版專屬）沒開 Code execution 開關**：用 claude.ai 免費版的人，要先在設定裡開啟「**Code execution and file creation**」，否則技能上傳了也**完全沒反應**。這是免費版「裝了沒動靜」的頭號原因，詳見 [免費版專區](free-version/) 的說明。

### Q：打開課程 repo 連結是 404（找不到頁面）？

代表你還沒登入、或邀請還在生效中。先確認你**用購買時提供的 GitHub 帳號登入**；若仍是 404，寄信到 service@eleanorfilm.academy 或加我們官方 LINE（附上你的 GitHub 帳號名稱），我們馬上幫你補邀請。

### Q：怎麼更新團隊到最新版？

不用重新下載安裝。想更新時，直接跟 Nora 說一句「**更新團隊**」，她會帶你把五位夥伴更新到最新版，**你的品牌資料、Token、個人設定都不會被動到**。詳見 [README 的「一句話更新」](README.md)。

### Q：不想要了，怎麼把團隊從電腦移除？

我們有準備**一鍵刪除碼**，複製貼上就能乾淨移除（只刪這個團隊、不會動到你 Claude 的其他東西）。見 [INSTALL.md 的「🗑️ 一鍵刪除碼」](INSTALL.md)。

## 🆘 卡關自救

### Q：如果遇到問題怎麼辦？

**第一時間，先把問題丟給你自己的 Claude（Nora 或對應的夥伴）去解決。** 把錯誤訊息、你在做什麼、卡在哪，原原本本貼給它，請它幫你診斷。

這樣做有兩個好處：

- **你的團隊會越來越強大**——每次一起解決問題，它就更懂你的狀況、更貼近你的品牌需求，下次更順手。
- **多數問題它當下就能幫你解掉**，比等人回覆更快。

如果真的試過還是解不了，或是遇到**技術上的問題**，再來找我們，我們很樂意協助你：

- 📧 **Email**：service@eleanorfilm.academy
- 💬 **官方 LINE**：[@eleanorfilmacademy](https://eleanorfilmacademy.tw/k0AWB)

> 來信／來訊時，附上**錯誤畫面截圖**＋**你做了哪些步驟**，我們能更快幫你定位問題。

### Q：Claude 突然當機、卡住、轉圈圈不回應，怎麼自救？

別緊張，先照這幾招試，多數情況馬上就好：

1. **換一個模型**：這是最常見的解法。模型偶爾會忙線或不穩，切換到另一個模型通常立刻恢復。
   - Claude Code：輸入 `/model` 切換成另一個模型再試。
   - 網頁版：在對話框上方／模型選單換一個模型。
2. **開一個新對話重試**：對話太長、塞太多內容時容易卡，開新對話往往就順了。
3. **完全關閉 Claude Code（或重新整理網頁）再打開**：清掉暫時的卡頓。
4. **內容太長就分段給**：一次貼太多字會超出負荷，拆成幾段、或改用新對話。
5. **可能是網路或用量上限**：檢查網路；若是免費版／當期額度用完，等額度恢復或升級後再試。

> 小提醒：當機通常是**暫時**的，不是你的團隊壞掉了。換模型 + 開新對話，九成都能解決。真的不行再依上一題找我們。

## ⚙️ 自動化與平台串接

### Q：我照做了，為什麼 Meta 廣告數據還是拉不到？

這不是你做錯。Meta 官方那個「自動拉廣告數據」的免費工具是**全球分批開放**的，台灣的廣告帳號很多還沒輪到，沒輪到就是接不上——這是 Meta 那邊的進度，急不來。在那之前完全不影響你用 Jack：把後台數據**貼給他**，他一樣幫你分析該加碼、暫停還是換素材，結果一樣，只是少了「自動」那一步。

### Q：Google 日曆串接看起來好難，一定要弄嗎？

不一定，這是全課**最進階、可以直接跳過**的一段。沒串接也不影響——Nora 一樣會幫你把這週該做的事列成清單，你自己貼進日曆就好。等哪天真的覺得手動貼有點煩，再回去照教學串也不遲。

### Q：我沒有 WordPress，Iris 還能幫我自動發文章嗎？

自動發文那段是給有 WordPress 的人用的。如果你用的是 Kajabi、Teachify、Wix 這類平台（多數台灣賣課的人都是），Iris 一樣幫你把整篇文章寫好，你**複製貼上、發布**，大概 2～3 分鐘。內容品質完全一樣，只是最後一步手動。

### Q：Nora 不是會自動追進度、提醒我嗎？為什麼都要我先開口？

對，預設是「**你開口，她才動**」——你問「進度怎麼樣」她就盤點給你。真正「每天早上自動跳出來提醒、推到手機」那種，要額外接排程工具＋ Claude 付費版才做得到，不是裝好就會自己跑。最省力的用法：養成每天開個對話跟 Nora 說一句「今天有什麼要盯的」，效果一樣。

### Q：為什麼裝好了還不能自動發文、自動拉數據？

團隊的「**腦**」（想策略、寫文案、做規劃）裝好就完整能用；但「**手腳**」（自動發文、自動拉數據）屬於進階，需要 Claude 付費版＋你自己去接一下平台，部分還要看平台對台灣開放沒。先用「腦」把內容做出來、手動發，流程跑順了再接自動化也不遲。

### Q：（免費版）怎麼讓 Claude 跨對話記得我的品牌？

免費版的品牌資訊是記在「**這個對話**」裡的，所以同一週的工作盡量待在同一個對話完成最穩。真的要開新對話前，先請 Nora「**幫我把品牌摘要列出來**」，複製貼到新對話開頭，她就能無縫接上、不用重新訪談你。（claude.ai 設定裡若有「Memory／個人化記憶」開關，打開也能跨對話記住大方向。）
