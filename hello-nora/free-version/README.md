## 開始之前：先打開一個免費開關（很重要，30 秒）

claude.ai 的「Skills（技能）」功能，**需要先開啟「Code execution」才會運作**——這個開關**免費版本來就免費可以開**，不是付費功能。

**做一次就好：**
1. claude.ai → 左下角你的頭像 → **設定（Settings）**
2. 進 **Capabilities（功能）**
3. 把 **「Code execution and file creation（程式碼執行與檔案建立）」打開**

> ⚠️ **這步沒開，等一下技能上傳了也不會有反應**——這是「裝了卻用不動」最常見的原因。先開好再往下走

## 第一步：裝技能（下載 → 上傳 → 開啟，三步）

1. 下載 `nora.zip`（先不用解壓縮，保持 zip）
2. claude.ai → 設定 → **Customize → Skills** → **上傳技能（Upload skill）**
3. 選 `nora.zip` 上傳 → 列表出現「hello-nora」→ **把開關打開**

其他四個一樣：下載 → 上傳 → 開啟。

## 第二步：跟 Nora 打招呼，讓她訪談你

✏️ 開新對話，輸入：

```
/hello-nora 你好
```

Nora 會自我介紹，然後**反過來訪談你**——問你做什麼、賣給誰、想給人什麼感覺、語氣偏好。你用講的回就好，她幫你把品牌記在這個對話裡。

> 💡 **免費版重點：品牌資訊存在「對話」裡，不是檔案。** 所以盡量**同一週的工作待在同一個對話**完成，Nora 才記得你。開新對話＝重新自我介紹（claude.ai 設定裡若有「Memory／個人化記憶」就打開，跨對話也記得大方向；沒有也沒關係，用上面那招貼品牌摘要最穩）。
>
> ⚠️ **真的要開新對話前**（例如當機卡住、想重來）：先請 Nora「**幫我把品牌摘要列出來**」，複製貼到新對話開頭，她就能無縫接上、不用重新訪談你。

## 第三步：一句話派工，team 動起來

✏️ 你只跟 Nora 說話，她會自動找對的人：

```
/hello-nora 幫我排這週三則 IG 貼文，主題是 ___
```

Nora 判斷這是社群的事 → 交給 Maya → 當場生出貼文（文案 + 時段 + hashtag + 配圖描述）。

免費版沒有自動發文，但只差一個動作：**複製 → 貼到 FB/IG → 發布**。

> 對剛起步的你，這樣反而最穩——發出前再用自己的眼睛確認一次。

## 免費版 vs Pro 差在哪？

**免費版該有的「腦力」一個都不少**——五位夥伴都能裝能用，策略、文案、方向都幫你做好。

差別只有一個詞：**自動化**。
- 免費版：Maya 寫好貼文，你複製貼上發
- Pro 版：接上工具後，連發文都幫你排好

先用免費版把流程跑順、團隊用熟。等哪天「手動貼上有點花時間」，再升級 Pro 也不遲。

## 卡關怎麼辦

看到問題，**先問 Claude**：把錯誤貼給它，問「這個錯誤為什麼？幫我修正」。

兩個最常見：
- 技能裝了沒反應 → **先確認「Code execution」開關有開**（設定 → Capabilities），這是最常見原因；再確認該技能本身的開關也打開（設定 → Skills，藍色）
- 跳出已達使用上限 → 等額度重置（建議一次把一週貼文請 Maya 生完，最省）

真的卡住 → LINE 官方帳號 **@eleanorfilmacademy** 或 service@eleanorfilm.academy
