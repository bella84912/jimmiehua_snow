## Cloudinary 圖片托管

> 🆓 **免費版看這裡**：下面「自動上傳」和存 API Secret 那套是**付費版（Claude Code）**才有。免費版直接用 **方式 B：網頁手動上傳**（到 cloudinary.com 點 Upload、複製 URL）就能拿到公開連結，一樣好用，不用設定 API。

圖片生好之後，需要一個**公開的 URL** 才能用在 IG 發文、SEO 文章、落地頁等地方。Cloudinary 是這個環節的標準工具。

### 為什麼需要 Cloudinary？

ChatGPT 生出來的圖片只能下載，沒有永久的公開連結。Cloudinary 幫你把圖片放到雲端，給你一個永久有效的 URL，可以直接貼到任何地方使用。

### 第一次設定（5 分鐘）

1. 打開 [cloudinary.com/users/register_free](https://cloudinary.com/users/register_free)
2. 免費註冊（25GB 免費額度，夠用）
3. 登入後進 **Dashboard**，找到這三個資訊：
   - **Cloud Name**（你的雲端名稱）
   - **API Key**（一串數字）
   - **API Secret**（32 位數字母）

> ⚠️ API Secret 不要給任何人，存在你自己的設定檔就好。

4. 把這三個資訊貼給 Leon（或直接存進 `~/.claude/skills/_brand/meta-config.md`）

### 日常使用方式

**方式 A：拖圖給 Claude Code（自動上傳）**

把圖片直接拖進 Claude Code 對話框 → Leon 自動上傳到 Cloudinary → 回傳公開 URL

```
你：（拖入圖片）這張幫我上傳到 Cloudinary
Leon：✅ 上傳完成！URL：https://res.cloudinary.com/你的CloudName/image/upload/v.../圖片名稱.jpg
```

**方式 B：網頁手動上傳**

1. 打開 [cloudinary.com](https://cloudinary.com) → 登入 → Media Library
2. 點「Upload」→ 拖入圖片
3. 上傳完成後，對圖片點右鍵 → Copy URL

### Cloudinary URL 長這樣

```
https://res.cloudinary.com/【CloudName】/image/upload/v1234567890/【檔名】.jpg
```

這個 URL 可以直接用在：
- IG 發文（圖片來源）
- SEO 文章配圖
- 落地頁圖片
- 任何需要圖片連結的地方
