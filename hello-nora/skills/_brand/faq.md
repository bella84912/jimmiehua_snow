# 常見問題與罐頭回覆

> Maya 回覆留言 / 私訊時，先讀這份檔案。
> 如果留言問的問題在這裡有答案，就用這裡的標準回覆（可依語氣微調）。
> 沒有對應答案的，再依品牌語氣自己擬，並建議你把新問題補進這份檔案。

---

## 使用方式

- 每組 = 觸發關鍵字 + 標準回覆
- 一個問題可以有多個觸發詞（用 `/` 分隔）
- 回覆裡的 `[連結]`、`[價格]` 等，請填成實際內容

---

## 課程相關

**Q：價格 / 多少錢 / 費用 / 怎麼收費**
A：（範例）Hi！課程資訊和最新優惠都在這邊 👉 [課程連結]，有任何問題都可以再問我們 😊

**Q：怎麼上課 / 上課方式 / 線上還是實體**
A：（範例）我們是線上課程，購買後就能無限次觀看，自己安排時間學習 🎬

**Q：有沒有期限 / 看多久 / 觀看期限**
A：（範例）購買後可以永久觀看，不用擔心看不完 👍

**Q：適合新手嗎 / 沒基礎可以嗎 / 零基礎**
A：（範例）完全適合！課程從最基礎開始教，一支手機就能上手 📱

**Q：有沒有優惠 / 折扣 / 折扣碼**
A：（範例）目前有限時優惠 👉 [連結]，把握時間～

## 不回覆 / 隱藏的類型

- 垃圾留言（加 LINE、賣東西、貼無關連結）→ 不回覆，建議隱藏
- 惡意攻擊 / 引戰 → 不回覆，視情況隱藏
- 純表情符號 / 純標記朋友 → 可選擇回愛心或不回
