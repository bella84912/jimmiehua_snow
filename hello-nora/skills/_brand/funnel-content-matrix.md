# 行銷漏斗 × 素材形式對照表

> 💡 **進階參考表，新手可先跳過。** 這份主要是 Maya／Jack 內部判斷用的對照邏輯，你**不需要手動填**——團隊會自己參考。等你想更懂「不同受眾要餵不同內容」時再回來看。

> **Maya 排內容、Jack 規劃廣告時，都要同時考量「受眾在哪一層」+「該用什麼形式」。**
> 不同受眾階層，要的內容和形式都不一樣——同一個主題，給冷流量和熱流量要做成不同的素材。

---

## 核心原則

```
受眾溫度決定「說什麼」+「用什麼形式」
冷流量 → 給價值、引好奇（不賣）
暖流量 → 建信任、消疑慮（軟轉換）
熱流量 → 催行動、給誘因（直接賣）
```

---

## TOF｜冷流量（認知層）

**對象：** 陌生人，還不認識你
**目標：** 停下來、記住你。**不賣東西**，先給價值或引發好奇
**廣告優化目標：** 觸及 / 影片觀看
**受眾設定：** 興趣受眾 / Lookalike
**⚠️ 不放連結 CTA**

| 形式 | 內容方向 | 重點 |
|------|---------|------|
| **直式影片 9:16**（Reels 15–30秒） | 教學乾貨、反差開場、UGC 自白 | 前 3 秒不提品牌名，靠內容吸引停留 |
| **橫式影片 16:9**（FB 30–60秒） | 教學示範、前後對比 | 有更多時間說故事 |
| **單圖** | 資訊圖、痛點金句圖 | 一句話點痛點，成本低、好測 hook |
| **輪播** | 「5 個你不知道的 XX 錯誤」 | 每張一重點，滑到最後才露品牌，好奇驅動滑動 |

---

## MOF｜暖流量（考慮層）

**對象：** 知道你，但還在觀望
**目標：** 建立信任、消除疑慮，讓他覺得「這個值得」
**廣告優化目標：** 流量 / 潛在客戶
**受眾設定：** 網站訪客 / 影片觀看者再行銷

| 形式 | 內容方向 | 重點 |
|------|---------|------|
| **直式影片 9:16**（30–60秒） | 學員見證、成果展示 | 真實學員、不要演員。口說成果 + 畫面佐證，最後帶課程名 |
| **輪播** | 課程內容介紹 / 學員 before-after | 每張一個模組或一位學員成果，最後一張放 CTA 連結 |
| **單圖** | 社會證明：學員人數、評價截圖 | 「已有 XXX 位學員」★★★★★，搭學員照片增可信度 |
| **橫式影片 16:9**（60–90秒） | 老師出鏡說明課程與差異化 | 適合放落地頁或搭再行銷 |

---

## BOF｜熱流量（行動層）

**對象：** 快決定了，給他最後一推
**目標：** 立刻購買或報名。緊迫感 + 誘因 + 明確 CTA
**廣告優化目標：** 轉換 / 購買
**受眾設定：** 加入購物車未結帳 / 落地頁訪客
**⚠️ 必放價格與 CTA**

| 形式 | 內容方向 | 重點 |
|------|---------|------|
| **單圖** | 限時優惠圖、早鳥倒數 | 「早鳥剩 3 天」「名額剩 XX」，大字價格 + 明顯 CTA 按鈕 |
| **直式影片 9:16**（15秒） | 快速 remind，直接說優惠 | 「你上次看過這堂課，早鳥快截止了」針對看過的人，不再介紹，直接催 |
| **輪播** | 課程內容 + 價格 + FAQ 一次解決 | 痛點→模組成果→價格 CTA，把猶豫原因全擊破 |
| **限動 / 直式靜態圖** | 倒數計時感設計 | 「今晚 12 點截止」+ 視覺倒數，點擊直達結帳頁 |

## Maya / Jack 使用方式

**Maya（社群排程）：**
- 排內容時判斷這篇是要養冷流量、暖流量、還是收割熱流量
- 依層級選對的形式（冷流量別硬塞 CTA、熱流量別只講故事不給價格）

**Jack（廣告規劃）：**
- 規劃廣告漏斗時，每一層用對應的形式 + 優化目標 + 受眾設定
- 同一個產品，三層各做對應素材（冷流量教學片 → 暖流量見證 → 熱流量優惠圖）
- 配合 `jack-ads` 的五大影片腳本架構（痛點/UGC/教學/反差/見證）對應使用

**共同原則：** 先問「這是給哪一層的人看的？」再決定內容與形式，不要一套素材打到底。
