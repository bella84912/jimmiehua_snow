# 把 FAQ 轉成 LINE「自動回應」關鍵字格式

> 學員有 LINE OA 就能用 LINE 內建的「自動回應」——跑在 LINE 雲端，**免伺服器、電腦關了照樣 24h 自動回**。
> Maya 把 `_brand/faq.md` 轉成 LINE 關鍵字格式，學員貼進 LINE 後台即可。

---

## 流程

1. Maya 讀 `~/.claude/skills/_brand/faq.md`
2. 把每組「Q（觸發關鍵字）+ A（回覆）」轉成 LINE 關鍵字回應格式
3. 產出清單給學員，學員貼進 LINE OA 後台

---

## LINE 後台設定位置

```
LINE Official Account Manager → 自動回應訊息
→ 「關鍵字回應」→ 新增
→ 關鍵字（可多個）+ 回應內容
```

> LINE 關鍵字回應是「**包含比對**」——客人訊息裡**只要出現**關鍵字就會觸發，不用一字不差。
> 所以每個問題設**幾個核心詞**就好（例：價格、費用、多少錢），不必窮舉所有說法。

---

## 轉換產出格式（Maya 給學員的）

```
【LINE 自動回應設定清單】

① 詢價
   關鍵字：價格、多少錢、費用、怎麼收費、報價
   回應：Hi！課程資訊和最新優惠都在這 👉 [課程連結]，有問題都可以問我們 😊

② 上課方式
   關鍵字：怎麼上課、上課方式、線上嗎、實體嗎
   回應：我們是線上課程，購買後無限次觀看，自己安排時間學習 🎬

③ 適合新手
   關鍵字：新手、零基礎、沒基礎、適合嗎
   回應：完全適合！從最基礎開始教，一支手機就能上手 📱

（依 faq.md 內容全部轉出）
```

---

## 提醒學員

- **關鍵字設核心詞即可**：LINE 是包含比對（訊息含關鍵字就觸發），抓幾個核心詞就好、不必窮舉；真的很冷門的問法再補上
- **設「一般回應」當後盾**：沒命中任何關鍵字時的預設回覆，例：「謝謝你的訊息！我們會盡快回覆你 🙌（或直接看課程 👉 [連結]）」
- **LINE 也有「AI 自動回應」**（智慧比對意圖，比關鍵字聰明）——付費方案才有，免費版用關鍵字回應即可
- FAQ 有更新 → 回來叫 Maya 重新轉一份，再貼進 LINE
