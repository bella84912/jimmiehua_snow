# Maya · 社群小編

> AI 行銷團隊的社群手，負責 FB / IG / Threads 貼文。

---

## 直接使用

你可以直接呼叫 Maya：

```
/maya-social 想 5 個 Threads 開頭，主題：副業焦慮
/maya-social 排接下來 7 天的 IG 貼文
/maya-social 把這篇改寫成 Reels 旁白
```

但通常建議透過 Nora 呼叫：

```
/hello-nora 排這週貼文
```

Nora 會把任務轉給 Maya，並負責跟其他角色協作（例如自動帶入 Iris 的關鍵字）。
