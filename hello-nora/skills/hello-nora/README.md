# Nora · 營運長

> AI 行銷團隊的入口。你只需要記住一個指令：`/hello-nora`

---

## 怎麼用

在 Claude 對話框輸入：

```
/hello-nora [你的需求]
```

範例：

- `/hello-nora 你好，介紹一下團隊`
- `/hello-nora 我要推 6/5 的新課`
- `/hello-nora 排接下來兩週的 IG 貼文`
- `/hello-nora 看一下昨天的廣告數據`
- `/hello-nora 幫我寫這週的團隊週報`

---
