## 0. 行銷規劃主軸流程（SOSTAC 簡化版）

**這是 Nora 每次幫學員做行銷規劃時的標準順序。不能跳過，也不能倒著來。**

> ⚠️ 學員是新手時，用白話文提問，不要說「SOSTAC」「現況分析」這種術語。

### 六個問題，依序問

| 步驟 | 專業說法 | 白話文問法 |
|------|---------|-----------|
| 1 | 現況分析（Situation） | 「你現在的狀況怎麼樣？有什麼困難？」 |
| 2 | 行銷目標（Objectives） | 「你希望這次達到什麼？多少訂單、多少流量？」 |
| 3 | 行銷策略（Strategy） | 「你打算主打哪種客群？用什麼方式接觸他們？」 |
| 4 | 行銷戰術（Tactics） | 「要用哪些管道？FB、IG、SEO、還是廣告？」 |
| 5 | 行銷計畫（Actions） | 「具體要做哪些事？誰來做、何時完成？」 |
| 6 | 行銷管理（Control） | 「怎麼追蹤有沒有效？看哪些數字？」 |

### 現況分析要問的事

```
搜尋行為：
  - 客人現在怎麼找到你的？（Google / IG / 朋友介紹）
  - 他們搜尋什麼關鍵字？

客戶關係：
  - 你的客人是誰？（基本背景、目標、痛點）
  - 他們進站後做了什麼？沒買的卡在哪裡？
  - 已經買的客人，買完之後有回購嗎？

現況困難：
  - 現在最大的問題是什麼？
    （沒流量 / 有流量沒轉換 / 舊客不回購 / 不知道怎麼開始）
```

### Nora 根據現況，決定派誰去做什麼

| 現況問題 | 優先處理 | 派給誰 |
|---------|---------|--------|
| 沒有人知道我 | 社群曝光、廣告 | Maya + Jack |
| 有流量但沒人買 | 優化 LP、強化 USP | Leon |
| 沒有關鍵字排名 | SEO 關鍵字 | Iris |
| 舊客不回購 | EDM / 社群互動 | Maya |
| 不知道從哪裡開始 | 先做現況分析 + 設目標 | Nora 帶著走 |

### 對新手學員的說話方式

- ✅ 「先跟我說一下，你現在的狀況是什麼？有什麼卡住了嗎？」
- ✅ 「你這次想達到什麼？賣多少、還是讓更多人認識你？」
- ❌ 「我們需要進行 SOSTAC 的 Situation Analysis」
- ❌ 「請填寫你的 KPI 和 OKR」

## 1. Persona 模板

**用途：** 確認目標客群前，先把 Persona 建好。沒有 Persona，所有行銷都是瞎猜。

### 填寫欄位
- 姓名、年紀、職業、居住地區、支付意願

### 精煉出價值主張（固定公式）
```
我們的＿＿＿＿（產品或服務）
協助＿＿＿＿＿（目標客群）
他想要＿＿＿＿（想完成的任務）
做法是＿＿＿＿（降低或避免痛點）
和＿＿＿＿＿＿（提升獲益）
```

## 9. Marketing ROI 計算公式

**用途：** 評估行銷投資是否值得。

### 公式
```
Marketing ROI = (營收 × 毛利率 − 行銷費用) / 行銷費用 × 100%
```

### 拆解公式
```
營收 = 流量 × 轉換率 × 客單價
流量 = 曝光量 × 點擊率
曝光量 = 自然曝光 + 付費曝光
```

### 提升 ROI 的 5 種策略
1. **降價**，大幅增加銷售量
2. **維持原價**，增加銷售量（優化文案素材 + SEO + 提高廣告預算）
3. **漲價**，維持銷售量（培養忠誠客戶）
4. **壓低進貨成本**（提高叫貨量、降低生產成本）
5. **降低變動費**（提高自然流量比例）

## 12. USP 發想框架（5W1H）

| 問題 | 說明 |
|------|------|
| **Who** | 誰是產品的目標消費者？ |
| **When** | 在什麼時間點需要？ |
| **Where** | 在什麼地點需要？ |
| **Why** | 為什麼一定要用我的產品，而不是競品？ |
| **What** | 我能解決目標消費者什麼問題／痛點？ |
| **How** | 解決完痛點之後，會有怎樣的心情／好處？ |

**USP 判斷標準：**
- 獨特：競爭對手沒有、或無法提供的
- 銷售力：讓消費者做出購買行動
- 主張：對消費者有足夠重要性，解決痛點的承諾

## 16. 完整電商營收公式

```
電商營收 = 新客營收 + 舊客營收

新客營收 = 流量 × 轉換率 × 客單價
舊客營收 = 客數 × 回購率 × 客單價
```

### 三大成本

| 成本 | 說明 |
|------|------|
| **流量獲取成本 CPC** | 每一次點擊花多少錢 |
| **名單獲取成本 CPA** | 取得一個潛在客戶花多少錢 |
| **訂單獲取成本** | 成交一筆訂單的總成本 |

### 五大指標（必追蹤）

| 指標 | 說明 |
|------|------|
| 曝光數 | 廣告被看到幾次 |
| 點擊率 | 看到廣告的人有多少比例點進來 |
| 轉換率 | 進站的人有多少比例實際下單 |
| 客單價 | 平均每筆訂單金額 |
| 客數 | 實際購買的人數 |

## 17. 流量獲取四步驟

```
Step 1 → 計算現有流量（搞清楚現在從哪裡來的人）
```

| 自然流量 | SEO、社群貼文 |

## 18. KOL / KOC / 網紅三種類型

| 類型 | 粉絲規模 | 特性 | 適合用途 |
|------|---------|------|---------|
| **KOC**（Key Opinion Consumer） | 1,000–5,000 | 一般消費者分享真實心得，信任感高 | 口碑擴散、真實使用心得 |
| **KOL**（Key Opinion Leader） | 1萬以上 | 在特定領域有影響力 | 品牌曝光、快速觸及新受眾 |
| **網紅** | 粉絲數大 | 知名度高，合作費用高 | 大型活動曝光，短期衝量 |

**選擇原則：**
- 預算少 → 多找 KOC，真實分享效果好
- 要快速曝光 → 找 KOL
- KOL 合作前確認：受眾是否吻合、互動率是否真實

## 20. 集客式行銷核心（SEO × 內容 × 社群）

**SEO 是讓廣告費越來越少的工具，不是快速獲利的工具。**

### SEO 優化三步驟

```
Step 1：用動態搜尋廣告找出能獲利的流量關鍵字
  → 先用廣告測試哪些關鍵字有人點、有人買

Step 2：改善網頁內容、提高關鍵字密度，降低 CPC
  → 把廣告有效的字，自然寫進網頁內容裡

Step 3：SEO 排名提升後，逐步取代付費廣告流量
  → 長期降低廣告依賴，自然流量佔比提高
```

### 集客三角

```
         SEO（長期）
        /           \
  社群行銷         內容行銷
（互動留客）    （吸引新流量）
        \           /
         部落格/文章
```

## 21. RFM 客戶分級模型

| 指標 | 說明 |
|------|------|
| **R（Recency）** | 最近一次購買是多久前？越近越好 |
| **F（Frequency）** | 總共買幾次？越多越好 |
| **M（Monetary）** | 總消費金額多少？越高越好 |

**應用方式：**
- 高 RFM → VIP 客，優先投資維繫關係
- 高 F 低 R → 曾經忠實，最近沒來，用 EDM / SMS 喚回
- 高 R 低 F → 新客，引導第二次購買
- 低 RFM → 睡眠客，低成本再行銷測試

## 22. 零售獲利公式

```
利潤 = 營收 - 成本

營收 = 流量 × 轉換率 × 客單價
成本 = 營業成本 + 營業費用
```

### 影響獲利的 5+1 關鍵

| 關鍵 | 可以怎麼影響 |
|------|------------|
| **市場規模** | 你選的市場夠大嗎？ |
| **流量** | 有多少人看到你？ |
| **轉換率** | 看到的人有多少比例買？ |
| **客單價** | 每次買多少錢？ |
| **成本** | 固定成本 + 變動成本控制 |
| **促銷**（+1） | 活動是否有效拉升毛利 |

| 類型 | 說明 | 策略 |
|------|------|------|
| **特色競爭者 Niche** | 在某個小眾領域很強 | 找他做不到的地方切入 |
| **關鍵競爭者 Key** | 直接搶你客人的 | 必須清楚跟他的差異化 |
| **成長競爭者 Growth** | 新興、快速成長中 | 注意趨勢，提早應對 |

### 品牌 3C 定位框架

```
    Consumer（消費者要什麼）
         ↙          ↘
Competitor         Company
（競爭者能給什麼）  （我能給什麼）

→ 三個圓圈的交集，就是你的差異化定位
```

## 24. 客戶完整旅程（從不認識到回購）

```
不知情
  ↓ 部落客行銷、Meta 廣告、KOL
認識品牌
  ↓ 內容行銷、SEO、Google Ads
考慮下單（新客到舊客的過渡）
  ↓ Landing Page、社證、再行銷
下單（臨門一腳）
  ↓ GDN 再行銷、限時優惠、口碑信任
購買
  ↓ 售後體驗、EDM、LINE
回購 / 會員經營
  ↓ RFM 分級、VIP 計畫、再行銷廣告
推薦
```

**每個階段要問的問題：**
- 客人在哪個階段卡住了？
- 這個階段最大的阻力是什麼？
- 要用什麼工具把他推到下一階段？
