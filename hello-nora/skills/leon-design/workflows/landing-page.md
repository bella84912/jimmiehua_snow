# Workflow：Landing Page 完整內容

> Leon 產出 LP 完整結構 + 文案 + 視覺骨架。

---

## 觸發條件

- 「做 Landing Page」
- 「LP 內容幫我寫」
- 「我要收名單，設計一個頁面」

---

## 執行步驟

### Step 1：確認活動目標

問用戶（一次問完）：

1. **這個 LP 的目的**？
   - 收 Email 名單
   - 預購 / 預約
   - 報名免費活動
   - 其他
2. **要承諾用戶什麼**？（你給的東西是什麼）
3. **有沒有時間壓力**？（截止日 / 限額）
4. **既有素材**：有沒有照片、影片、見證可以用？

### Step 2：讀資料

- `_brand/brand-guide.md`
- `_brand/voice-profile.md`
- `_brand/product-info.md`
- `_brand/icp.md`

### Step 4：一塊一塊做成真的網頁（HTML）

> ⚠️ 上面的內容稿只是「藍圖」，不是交付物。**我不會把文字稿丟給你就收工**——我直接在 Claude Code 裡把它做成一頁真的、有設計感、能部署的 HTML 網頁。

照 SKILL.md 的做法**一塊一塊帶著做**（不要一次吐一整頁陽春 HTML）：

1. 先給你看「整頁藍圖」（這頁會有哪些區塊），確認排序 OK
2. 從 Hero 開始，每塊照小循環：**寫真實文案 → 做成有質感的 HTML 區塊（排版層次、留白、品牌色、CTA 樣式）→ 給你看 + 問要不要調**
3. 你確認 → 累積到頁面 → 做下一塊
4. 全部做完 → 合成**完整單檔 HTML**（手機優先、套 `_brand/brand-guide.md` 色票），整體微調一次

### Step 5：預覽與上線

做好的是一個 `.html` 檔，存在你電腦裡。

- **想自己先看** → 雙擊那個 `.html` 檔，瀏覽器就打開了（不用工具、不用註冊、不用上網）
- **要正式上線給客人看** → 把這個 HTML 放上**你自己的平台**（Teachify／Kajabi／官網）。還沒有平台的話，這就是 Jack 流程裡「先建站」那一步（Strikingly／Carrd／Teachify 等），建好再把頁面放上去。

> 想改哪一段隨時跟我說，我改完你重新存一次檔、上傳更新就好。
