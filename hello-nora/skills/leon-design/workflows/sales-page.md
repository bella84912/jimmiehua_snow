# Workflow：Sales Page 完整內容

> Leon 產出 SP 完整結構 + 文案 + 視覺骨架。

---

## 觸發條件

- 「做 Sales Page」
- 「SP 內容幫我寫」
- 「我要推這個產品，幫我寫銷售頁」

---

## 執行步驟

### Step 1：確認產品資訊

從 `_brand/product-info.md` 讀產品資料。如果沒有或不完整，問用戶：

1. 產品名稱 + 定價（含原價 vs 優惠價）
2. 主要解決什麼問題
3. 給誰用（從 ICP 抓）
4. 有沒有保證 / 退款機制
5. 有沒有時間壓力（早鳥截止、限量）

### Step 2：產出 SP 完整內容

```markdown
# Sales Page｜[產品名稱]

## Hero 區（首屏）

### 大標題
[一句話說清楚「這是什麼」+「解決什麼」]

### 副標題
[對誰最有用 + 為什麼是現在]

### Hero CTA
[立即購買 / 立即加入 / 馬上開始]

### 視覺建議
[產品 mockup / 學員上課畫面 / 成果展示]

---

## 痛點放大區

### 你是不是這樣？
[3-5 個具體痛點場景，比 LP 更深更細]

### 如果繼續這樣下去
[不買的後果，講「未來會有多痛」]

### 視覺建議
[黑底白字 / 紅色強調，製造壓迫感]

---

## 價值錨點區（製造對比）

### 別人是這樣解決的
- 方法 A：[名稱]｜[價格]｜[缺點]
- 方法 B：[名稱]｜[價格]｜[缺點]

### 我們不一樣
[凸顯你的獨特解方，3 個差異點]

---

## 產品介紹（不只列功能，講「對你的意義」）

### 你會得到什麼

#### 模組 / 內容 1：[名稱]
**內容**：[功能描述]
**對你的意義**：[這代表你可以做到什麼]

#### 模組 / 內容 2：[名稱]
**內容**：...
**對你的意義**：...

[依此類推，建議 4-7 個模組]

### Bonus（附加價值）
- [Bonus 1]：價值 NT$XXX
- [Bonus 2]：價值 NT$XXX

---

## 學員見證（必備）

### 文字見證
- 「[詳細見證引文，最好有具體成果數字]」— [姓名 / 職業 / 照片]

### 影片見證（如果有）
[影片連結 + 重點截圖]

### 數據見證
- 累積 [X] 位學員
- [Y]% 學員推薦
- 平均 [Z] 個月看到成效

---

## 定價區（核心）

### 一般價 vs 你的價
**原價**：NT$XXXX
**現在的價**：NT$XXX

### 為什麼這個價值得
- 拆解：[1 天只要 NT$XX]
- 對比：[相當於上課 X 堂 1:1 諮詢]
- 限定：[早鳥優惠剩 X 天 / 限量 XX 名]

### 主 CTA
[立即購買 / 加入課程]

### 信任語
[100% 滿意保證 / X 天無條件退費 / 終身學習無期限]

---

## 保證機制

### 我們的承諾
[退費條件 + 流程，清楚到沒有模糊空間]

### 我們不適合誰
[反向篩選，反而提高信任]

---

## Q&A（破除購買障礙）

### 最常見的 5-8 題
[每題針對一個常見購買障礙]

範例必備題目：
- 「我沒時間怎麼辦？」
- 「我是新手能跟上嗎？」
- 「跟其他課差別？」
- 「不適合可以退嗎？」
- 「有沒有付款分期？」

---

## 倒數區（如果有時限）

[早鳥倒數 / 限量剩餘名額]

---

## Footer CTA（最後一擊）

### 標題
[最後一次強力呼籲]

### CTA Button
[同主 CTA]

### PS（補充說明）
[強化緊迫感或重申保證]
```

### Step 3：一塊一塊做成真的銷售頁（HTML）

> ⚠️ 上面的內容稿是「藍圖」，不是交付物。我直接在 Claude Code 裡把它做成一頁真的、有設計感、能部署的 HTML 銷售頁——**不是丟你一份 Markdown 就收工**。

照 SKILL.md 的做法**一塊一塊帶著做**（10 個區塊：Hero → 痛點放大 → 價值錨點 → 產品介紹 → 見證 → 定價 → 保證 → Q&A → 倒數 → Footer CTA）：

1. 先給你看整頁藍圖，確認排序 OK
2. 每塊照小循環：**寫真實文案 → 做成有質感的 HTML 區塊（痛點黑底壓迫感、定價大字對比劃線、倒數紅 banner、CTA 明顯）→ 給你看 + 問要不要調**
3. 你確認 → 累積到頁面 → 做下一塊
4. 全部做完 → 合成**完整單檔 HTML**（手機優先、套 `_brand/brand-guide.md` 色票），整體微調一次

### Step 4：預覽與上線

做好的是一個 `.html` 檔。

- **想自己先看** → 雙擊那個 `.html` 檔，瀏覽器就打開了（不用工具、不用註冊）
- **要正式開賣、掛廣告** → 把這個 HTML 放上**你自己的平台**（Teachify／Kajabi／官網）。還沒有平台的話，就是「先建站」那一步（Strikingly／Carrd／Teachify 等），建好再放上去。

> 做完銷售頁，下一步通常是開始打廣告 → 跟我說一聲，我把這份 SP 的重點交給 Jack，請他寫對應的廣告 hook。
