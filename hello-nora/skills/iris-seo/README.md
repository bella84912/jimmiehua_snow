# Iris · SEO 專員

> 每個月想做 SEO，跟 Nora 說一聲，Iris 就幫你抓 20 個高潛力關鍵字——評估競爭度、商業意圖、優先級，附 SEO 友善的文章大綱。
>
> **想要 Google 官方的真實月搜尋量？** 用課程教的 **Google 關鍵字規劃工具**查（官方、免費、查到飽），把數字貼回來給 Iris，她就用真數據幫你排序。沒查時 Iris 會用 AI 估算、並標明非真實數據。
>
> （想自動化、不想每次手動查的進階用戶，也可自行接第三方關鍵字 MCP 讓 Iris 直接抓——但那類服務多有免費額度上限、可能要錢，請自行評估，課程不特別推薦。）

---

## 直接使用

```
/iris-seo 找關鍵字，主題：攝影課
/iris-seo 拆解這篇競品文章 [URL]
/iris-seo 給我「剪映教學」的 SEO 大綱
```

或透過 Nora：

```
/hello-nora 我這月想做 SEO，主題：副業
```
