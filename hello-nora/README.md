✏️ **最簡單（推薦）**：登入 GitHub 打開 repo → 綠色「**Code**」→「**Download ZIP**」→ 解壓縮 → 把 **hello-nora 資料夾拖進 Claude Code 對話框**，貼這句：

> 「請把這個 hello-nora 資料夾裡 skills 底下的全部內容，複製到我的 Claude 技能資料夾（`.claude/skills/`，沒有就建立），包含 _brand 和所有子資料夾。複製完成後，**請確認 hello-nora、maya-social、leon-design、iris-seo、jack-ads 五位成員和 _brand 都到齊了並列給我看，少了就再複製一次。** 全部到齊後跟我說「🎉 安裝完成」，並提醒我完全關閉 Claude Code 再重新打開。」

裝好後**完全關閉 Claude Code 再重開**，輸入 `/hello-nora 你好` 就開始。

> **進階（給習慣指令的人）**：也可以一行裝完。⚠️ 這條 `git clone` 需要你的電腦**先完成 GitHub 命令列登入授權**（沒授權過會跳 `Authentication failed` 或一直要帳密）——不熟的話別硬試，**直接用上面的「Download ZIP」拖檔最穩**：
> ```bash
> cd ~/Desktop && git clone https://github.com/eleanorfilm-academy/hello-nora.git && mkdir -p ~/.claude/skills && for s in hello-nora/skills/*; do n=$(basename "$s"); if [ "$n" = "_brand" ] && [ -d ~/.claude/skills/_brand ]; then echo "  ⏭️  _brand 已存在，保留你的品牌資料、跳過不覆蓋"; else cp -r "$s" ~/.claude/skills/; fi; done && printf "\n🎉 恭喜！Hello Nora 行銷團隊安裝完成！請完全關閉 Claude Code 再重新打開，輸入 /hello-nora 你好 就能開始。\n\n"
> ```
> 不熟指令就用上面的「Download ZIP」最穩。完整步驟與防呆見 [INSTALL.md](INSTALL.md)。

## Claude 免費版 vs Claude 付費版

**免費版的「腦」一個都不少**——五位成員的問答、規劃、寫作、策略能力完整。差別只在「手腳」（自動化）：

| 能力 | Claude 免費版 | Claude 付費版（Claude Code）|
|------|:------:|:------:|
| 內容生產（文案、規劃、策略、SEO 文章…） | ✅ | ✅ |
| 五位成員的完整方法論 | ✅ | ✅ |
| 自動發文 / 拉廣告數據 / MCP 串接 | — | ✅ |
| 一鍵指令（/setup /report /deck…） | — | ✅ |

> 免費版：內容做好，你**手動複製貼上**發布。
> 付費版（Claude Code）：**接上工具後**可自動發文、自動拉數據。
> 先用免費版把流程跑順，需要自動化再升級 Claude Pro（約 US$20/月，實際以官方公告為準）。

> ⚠️ **「自動化」要先接、且看平台臉色，老實說在前面**：自動發 FB／IG 要走完 Unit 7 的串接（要 Facebook 粉專、複製貼上一些指令）；自動拉 **Meta 廣告數據**的官方免費工具目前**全球分批開放、台灣帳號可能還沒輪到**，沒輪到就先用手動貼數據；**Google 廣告**沒有官方一鍵免費版，門檻較高（見 `ADVANCED-google-ads.md`）。這些都不影響你用團隊的「腦」，只是「手腳」要不要接、接不接得上而已。

## 🔄 一句話更新，永遠用最新版

這個團隊會持續變強——我們會不斷幫每位夥伴新增能力、優化工作流。

**你不需要重新下載安裝。** 想更新時，只要跟 Nora 說一句話：

更新團隊
```

Nora 會帶你更新：
1. 從你已被邀請的私密 repo 重新抓最新版角色設定
2. 更新 Nora / Maya / Leon / Iris / Jack
3. 告訴你這次更新了什麼

> **你的資料完全不受影響**——品牌資料、Meta Token、個人稱呼設定都會原封不動保留，更新只會讓夥伴變更強。
>
> 💡 更新後，開一個新對話就會用到最新版。
>

> - **最穩（人人適用）**：回 repo 重新 **Download ZIP**，解壓後拖給 Claude，說「更新角色資料夾，`_brand/` 不要覆蓋」。
> - **當初用指令 clone 的人**：在 `~/Desktop/hello-nora` 跟 Nora 說「更新團隊」，她會用 `git pull` 拉最新版（需 GitHub 已授權）。
