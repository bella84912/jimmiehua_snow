## 你會得到什麼

Jack 用三層漏斗思維管理你的 Meta 廣告：
- **冷受眾（40%預算）**：陌生人，用勾人的 hook 先讓他認識你
- **暖受眾（50%預算）**：看過你內容的人，推深一點的訊息
- **熱受眾（10%預算）**：快要轉換的人，直接給他最強的理由

他有 **5 個技能點**，這個單元會帶你一個一個用過、每個都當場驗收：

- **5-1　廣告文案 / Hook**：5 種公式（痛點／好處／社證／敘事／爭議），並建議先測哪個
- **5-2　影片廣告腳本**：15 秒腳本，照五大架構寫
- **5-3　受眾 + 預算規劃**：用三層漏斗幫你分配預算、規劃廣告組合
- **5-4　廣告數據週報**：給加碼／暫停／換素材建議（數字貼給他就行；想自動拉 Meta 數據需先串接，門檻見 5-4）
- **5-5　兩階段投放**：沒投過也行——先了解你現況，再帶你規劃

> 想看 Jack 完整能耐？裝好後直接問他「**你會做什麼**」。

## Jack 已經裝好了

Jack 在 Unit 0 跟團隊一起裝好了，這個單元直接開始用他。

> 沒裝到？兩種方式任選：
> - **整團一起裝**（推薦）：回 Unit 0 的「一次裝好整個團隊」那步，一次把五位＋品牌資料＋指令全部裝齊。
> - **只裝 Jack 這一位**：照下面三步（僅供「只想先試一位」用，見下方提醒）。

### （備案）只單獨安裝 Jack

> ⚠️ **這個方式只會裝到 Jack 一個人**——不會裝到品牌資料（`_brand`）、`/setup` 等指令、工作流。要完整功能，請改用 Unit 0 的「一次裝好整個團隊」。

不想跑終端機、或只想先試 Jack 一個人，可以用「貼安裝檔」的方式：

1. 點開 **[👉 Jack 安裝檔](https://github.com/eleanorfilm-academy/hello-nora/blob/main/skills/jack-ads/SKILL.md)**
2. 瀏覽器會顯示一段純文字 → **全選複製**（Mac：`⌘A → ⌘C`；Windows：`Ctrl A → Ctrl C`）
3. 貼進 Claude Code 視窗，輸入：**「幫我安裝」**

Claude 會把檔案存到 `~/.claude/skills/jack-ads/`。**完全關閉 Claude Code 再重新打開**，Jack 就上線了。

## Jack 的 5 個技能點（跟著做，一項一驗收）

> 有在跑廣告先做 **5-4（數據週報）**；還沒跑就從 **5-1（Hook）** 或 **5-5（兩階段投放）** 開始。每一項都有明確的「✅ 驗收」。指令透過 Nora 轉給 Jack，想直接找他打 `/jack-ads`。

### 5-1　廣告文案 / Hook

✏️ 複製貼給 Claude：

```
/hello-nora 請 Jack 幫我寫 5 個廣告 hook，產品是 [你的產品]
```

Jack 用 5 種公式（痛點／好處／社證／敘事／爭議）給你 hook，並建議先測哪一個。

> ✅ **驗收**：你有 5 個不同角度的廣告 hook，知道先測哪個。

### 5-2　影片廣告腳本

✏️ 複製貼給 Claude：

```
/hello-nora 請 Jack 幫我寫一支 15 秒廣告腳本，產品是 [你的產品]
```

照五大架構（鉤子 → 痛點 → 方案 → 證明 → 行動）寫成可直接拍的腳本。

> ✅ **驗收**：你有一支 15 秒、分鏡清楚的廣告腳本。

### 5-3　受眾 + 預算規劃

✏️ 複製貼給 Claude：

```
/hello-nora 我有每天 [XX 元] 預算，請 Jack 幫我分配
```

Jack 用三層漏斗（冷 40%／暖 50%／熱 10%）幫你分配預算、建議開幾組廣告組合。

> ✅ **驗收**：你有一份「預算怎麼分、開幾組」的具體規劃。

### 5-4　廣告數據週報

✏️ 把廣告數據截圖或貼上數字：

```
/hello-nora 請 Jack 看這組數據，告訴我該怎麼做
[貼上你的數據]
```

Jack 幫你讀 CTR／CPM／ROAS，給你加碼／暫停／換素材的具體建議。**最簡單：把後台數據截圖或數字貼給他就行，不用串接任何東西。**

> 💡 **想讓他「自動拉數據」、不想每次手動貼？** 那要先串 Meta Ads MCP，門檻先講清楚（照 `/setup` 一步步帶你接）：
> - 需要 **Claude 付費版（Pro／Max）**，而且你是該 Meta 企業管理（Business Manager）的**管理員**。
> - 用的是 **Meta 官方免費連接器**（不用開發者權杖）；但官方正分批開放，**台灣帳號可能還沒輪到**——接了卻顯示「未啟用」不是你做錯，是還沒開放到你。
> - 還沒開放時可先用備案 **pipeboard**（第三方、能立刻用），但**免費方案每週只有 30 次查詢**，狂用要付費，先知道別撞牆。
> - 沒串也完全不影響——把數字貼給 Jack，他一樣幫你分析。

> ✅ **驗收**：你知道這組廣告該加碼、暫停、還是換素材。

### 5-5　兩階段投放（沒投過也能上手）

✏️ 複製貼給 Claude：

```
/hello-nora 我沒投過廣告，請 Jack 帶我規劃 [產品] 的第一波投放
```

Jack 先了解你的現況，再一步步帶你把第一波廣告規劃出來。

> ✅ **驗收**：你有一份新手版的第一波投放計畫，知道下一步做什麼。
