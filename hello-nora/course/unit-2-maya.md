## 你會得到什麼

一個叫做 **Maya** 的 AI 社群小編，負責 FB、IG、Threads、LINE 的內容。

她懂你的品牌語氣（從 Unit 0 填的資料來的），有 **6 個技能點**，這個單元會帶你一個一個用過、每個都當場驗收：

- **2-1　寫貼文 / Hook**：完整貼文或多個開頭，附 hashtag、配圖說明
- **2-2　排一週貼文行事曆**：7 天計畫，**對照台灣行銷節日**幫你借勢，標好每篇用圖文／輪播／短影音
- **2-3　一稿多發**：同主題針對 FB／IG／LINE 各裂變一版（不是複製貼上）
- **2-4　IG 輪播圖卡**：封面 Hook → 重點卡 → CTA，交給 Leon 出視覺
- **2-5　回留言 / 私訊**：依品牌語氣逐則建議回覆（留言貼／截圖給她即可）
- **2-6　FB 自動發文 / LINE 群發**：接好帳號後直接發布（用 `/setup` 串接，Meta 發文見 Unit 7）

> 想看 Maya 完整能耐？裝好後直接問她「**你會做什麼**」，她會用白話自我介紹。

## Maya 已經裝好了 ✅

Maya 在 Unit 0 跟團隊一起裝好了，這個單元直接開始用她。

### （備案）只單獨安裝 Maya

> ⚠️ **這個方式只會裝到 Maya 一個人**——不會裝到品牌資料（`_brand`）、`/setup` 等指令、工作流。要完整功能，請改用 Unit 0 的「一次裝好整個團隊」。
>
> ⚠️ **更重要的：Maya 的排程行事曆、一稿多發、小紅書／EDM 都要讀她自己的工作流檔，單貼一個 `SKILL.md` 這些檔不會跟著來——所以「Maya 本人」碰到進階任務也會做不完整（找不到檔）。這個備案只適合「先見一面、試試個性」；要正式用，請走 Unit 0 整團裝。**

不想跑終端機、或只想先試 Maya 一個人，可以用「貼安裝檔」的方式：

1. 點開 **[👉 Maya 安裝檔](https://github.com/eleanorfilm-academy/hello-nora/blob/main/skills/maya-social/SKILL.md)**
2. 瀏覽器會顯示一段純文字 → **全選複製**（Mac：`⌘A → ⌘C`；Windows：`Ctrl A → Ctrl C`）
3. 貼進 Claude Code 視窗，輸入：**「幫我安裝」**

Claude 會把檔案存到 `~/.claude/skills/maya-social/`。**完全關閉 Claude Code 再重新打開**，Maya 就上線了。

## Maya 的 6 個技能點（跟著做，一項一驗收）

> 不用一次全做完。先把 **2-1、2-2** 走過（最常用的兩個），其他有需要再回來用。每一項做完都有明確的「✅ 驗收」。
> 重新打開 Claude Code，下面每個指令都透過 Nora 轉給 Maya；想直接找她也可以打 `/maya-social`。

### 2-1　寫貼文 / Hook

✏️ 給你一次多個開頭讓你挑，或直接寫整篇。

```
/hello-nora 請 Maya 幫我寫 5 個 Threads hook，主題是 [你今天想說的事]
```

拿到 5 個風格不同的 hook，每個附「適合場景」「建議發文時間」；要整篇就說「幫我寫一篇○○貼文」，她會附 hashtag、配圖說明。

> ✅ **驗收**：你有 5 個能直接用的 hook，挑一個就能發。

### 2-2　排一週貼文行事曆

✏️ 7 天發什麼一次排好，對照台灣行銷節日幫你借勢。

```
/hello-nora 幫我排這週的貼文，主題是 [你的產品或活動]
```

她會給 7 天計畫，標好每篇用圖文／輪播／短影音。

> ✅ **驗收**：你手上有一份這週 7 天、標好形式的發文計畫。
> 💡 有裝 Google 日曆的話，加一句「排完直接推到我的 Google 日曆」，她確認後會一次推進你的日曆（進階選用，見上面 Step 1）。

### 2-3　一稿多發

✏️ 同一個主題，針對 FB／IG／LINE 各裂變一版——不是複製貼上。

```
/hello-nora 這個主題幫我發 FB、IG、LINE：[主題]
```

> ✅ **驗收**：你拿到同主題、三個平台各一版、語感不同的貼文。

### 2-4　IG 輪播圖卡

✏️ 把一篇內容拆成封面 Hook → 重點卡 → CTA，再交給 Leon 出視覺。

```
/hello-nora 請 Maya 把這篇做成 IG 輪播：[貼上內容]
```

> ✅ **驗收**：你有一組輪播每張卡的文案結構（接著請 Leon 出圖、自動存檔，見 Unit 3）。

### 2-5　回留言 / 私訊

✏️ 把留言貼或截圖給她，她依品牌語氣逐則建議回覆。

```
/hello-nora 請 Maya 幫我回這些留言：[貼上留言]
```

> ✅ **驗收**：你拿到每則留言的建議回覆，挑順眼的貼回去就好。
> 想讓她**自動抓下留言**再回，需先完成 Unit 7 的 Meta 串接。

### 2-6　FB 自動發文 / LINE 群發（需先串接）

接好帳號後，Maya 直接幫你發 FB 貼文、發 LINE 通知。先用 `/setup` 串接帳號（FB 發文走 Unit 7 的 Meta 串接）。

> ✅ **驗收**：帳號接好後，說「排程到粉專」「發 LINE 通知」就能直接發布。
