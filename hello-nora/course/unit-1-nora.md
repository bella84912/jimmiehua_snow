## 你會得到什麼

一個叫做 **Nora** 的 AI 營運長，住在你的 Claude 裡，有 **5 個技能點**，這個單元會帶你一個一個用過、每個都當場驗收：

- **1-1　任務拆解與派工**：你說「我要推新課」，她拆成一串任務、分派給對的夥伴
- **1-2　進度追蹤 + 團隊週報**：一句話請她彙整這週做了什麼、成效如何、下週優先做什麼
- **1-3　策略分析（看數據給方向）**：看你的現況和數據，告訴你接下來最該集中火力的幾件事
- **1-4　出客戶／老闆簡報**：用 🔒 `/deck` 把成果做成投影片（🔒 ＝需 Claude 付費版，免費版沒有這個斜線指令、改用講的）
- **1-5　防呆精靈幫你接平台**：用 🔒 `/setup` 一步步把 FB／IG／LINE／廣告接起來（🔒 ＝需 Claude 付費版）

你跟她說「我要推新課」，她會拆解任務、分派給對的夥伴、追蹤進度。
整個課程都是先裝她，因為她是你唯一需要開口說話的入口。

> 想看 Nora 完整能耐？裝好後直接問她「**你會做什麼**」。

## Nora 已經裝好了

你在 **Unit 0 的「一次裝好整個團隊」** 那步，已經把 Nora（和其他四位）一起裝好了，這個單元直接開始用她。

> 沒裝到？兩種方式任選：
> - **整團一起裝**（推薦）：回 Unit 0 的「一次裝好整個團隊」那步，一次把五位＋品牌資料＋指令全部裝齊。
> - **只裝 Nora 這一位**：照下面三步（僅供「只想先試一位」用，見下方提醒）。

### （備案）只單獨安裝 Nora

> ⚠️ **這個方式只會裝到 Nora 一個人**——不會裝到品牌資料（`_brand`）、`/setup` 等指令、工作流。要完整功能，請改用 Unit 0 的「一次裝好整個團隊」。
>
> ⚠️ **更重要的：Nora 的派工、週報、策略分析都要讀她自己的工作流檔，單貼一個 `SKILL.md` 這些檔不會跟著來——所以「Nora 本人」碰到進階任務也會做不完整（找不到檔）。這個備案只適合「先見一面、試試個性」；要正式用，請走 Unit 0 整團裝。**

不想跑終端機、或只想先試 Nora 一個人，可以用「貼安裝檔」的方式：

1. 點開 **[👉 Nora 安裝檔](https://github.com/eleanorfilm-academy/hello-nora/blob/main/skills/hello-nora/SKILL.md)**
2. 瀏覽器會顯示一段純文字 → **全選複製**（Mac：`⌘A → ⌘C`；Windows：`Ctrl A → Ctrl C`）
3. 貼進 Claude Code 視窗，輸入：**「幫我安裝」**

Claude 會把檔案存到 `~/.claude/skills/hello-nora/`。**完全關閉 Claude Code 再重新打開**，Nora 就上線了。

## Step 1：跟 Nora 打第一次招呼

✏️ 在 Claude Code 輸入：

```
/hello-nora 你好，自我介紹一下
```

Nora 會：
- 先讀取你在 Unit 0 建立的品牌資料
- 介紹自己跟團隊 4 個夥伴
- 告訴你品牌資料的完整度
- 問你今天想做什麼

## ✅ 驗收任務

✏️ 叫 Nora 做一件具體的事：

```
/hello-nora 我這週想推一個新產品，你覺得我應該先做什麼？
```

她應該給你**具體的下一步建議**——不是「視情況而定」這種模糊話，而是「先做 X，再做 Y」。
這樣就算過關。

## Nora 的 5 個技能點（跟著做，一項一驗收）

> 跟她打過招呼後，下面 5 件事就是 Nora 平常幫你做的。先把 **1-1、1-3** 走過（最常用），其他有需要再回來用。每一項都有明確的「✅ 驗收」。
> 直接打 `/hello-nora` 加你的需求就行；用白話講也通。

### 1-1　任務拆解與派工

✏️ 一句模糊的需求，她拆成一串「誰做、什麼時候做、誰先誰後」的任務鏈。

```
/hello-nora 我 6/5 要推新課，定價 NT$720 預購，幫我搞定
```

她會給你一份倒推時程（T-14 預熱、T-0 開賣…）、把任務分派給 Iris／Leon／Maya／Jack，標好依賴關係，最後給你 3 個下一步選項（照順序跑／同時跑／只做最重要的）。

> ✅ **驗收**：你拿到一份「時程＋任務分派＋誰先做」的作戰表，挑一個下一步就能開工。

### 1-2　進度追蹤 + 團隊週報

✏️ 一句話請她把一週的成果收成一頁式週報。

```
/hello-nora 幫我整理這週的行銷週報
```

她會彙整各角色做了什麼、數字摘要、本週亮點與警訊、下週 3 件重點。

> ⚠️ 數字部分她只填**你貼給她**、或**有接數據才拉得到**的真實數字，沒有的欄位會請你補、不會自己編。自動拉廣告／流量數據需 Claude 付費版＋串接（見 Unit 5、7）。

> ✅ **驗收**：你拿到一份一頁式週報，看得到這週做了什麼、下週該做什麼。

### 1-3　策略分析（看數據給方向）

✏️ 不確定下一步時，請她綜合你的現況給方向。

```
/hello-nora 我接下來最該做什麼？給我策略建議
```

她會把你的商品、客群、做過的事、廣告數據（有接的話）綜合起來，給你「現在最該集中火力的幾件事」——是具體建議，不是流水帳。（付費版可直接打 `/report`。）

> ✅ **驗收**：你拿到 3 件以內、有理由的「現在該做的事」，而不是「看情況」。

### 1-4　出客戶／老闆簡報（需 Claude 付費版）

✏️ 把成果做成給老闆或客戶看的成效簡報。

```
/deck
```

她會把數據和成果做成品牌風格的 HTML 投影片，可一鍵存成 PDF（要可編輯的 PPT 需另裝工具、版面再微調）。

> ✅ **驗收**：你拿到一份能直接交出去的成效簡報（可存 PDF）。
> 🆓 免費版沒有 `/deck` 指令；可以請 Nora 把成果整理成文字版，自己貼進簡報工具。

### 1-5　防呆精靈幫你接平台（需 Claude 付費版）

✏️ 想讓團隊「自動發文／自動拉數據」，先用設定精靈把帳號接起來。

```
/setup
```

她會像真人客服一樣，一次只帶你做一個動作、每貼一個東西就當場驗證對不對、卡住就看你的截圖告訴你點哪裡。

> ✅ **驗收**：跟著精靈把至少一個平台（FB／IG／LINE／廣告）接起來、當場驗證成功。
> 🆓 免費版不需要也不能用 `/setup`（網頁版接不了），內容手動複製貼上發布即可。
