# 🎁 30 個實戰 Prompt 食譜（早鳥 BONUS · 複製貼上就能用）

> 這不是「課程重點複習」，而是 30 個**真實情境的組合技食譜**——常常一句話就讓 Nora 同時調度好幾位夥伴，把進階的打法直接打包給你。
> 把 `[方括號]` 換成你的內容，貼進 Claude 就會動。挑你這陣子正要做的那件事，從那一條開始。

**怎麼用**

- **付費版（Claude Code）**：整句連 `/hello-nora` 一起貼。
- **免費版（claude.ai 網頁）**：把開頭 `/hello-nora` 拿掉，直接講後面的需求即可（多夥伴協作一樣會幫你做，只是要你手動把成品複製出去用）。
- 方括號 `[ ]` 是要你換成自己內容的地方；不確定怎麼填，貼上去後團隊會反問你。

## 🚀 上市與促銷

**1. 新品上市 7 天衝刺**
```
/hello-nora 我 [日期] 要上 [產品／課程]，幫我排一份 7 天上市衝刺計畫：Maya 排每天的貼文、Leon 出銷售頁和主視覺、Jack 規劃廣告預算與 hook、Iris 補一篇能被搜尋到的文章，最後彙整成一張我照著做就行的時間表。
```

**2. 限時優惠活動懶人包**
```
/hello-nora 我要辦一檔 [優惠內容，例：限時 8 折 3 天] 的活動，幫我一次備好：Leon 出活動頁、Maya 出開賣／提醒／最後一天三波貼文、Jack 出一組導購廣告文案。
```

**3. 倒數催單組合包**
```
/hello-nora 我的 [產品] 剩 [X] 天就漲價／截止，幫我做一套倒數催單：Maya 出多平台倒數貼文、Jack 出一支主打急迫感但不討人厭的廣告 hook。
```

**4. 線上活動／直播完整宣傳**
```
/hello-nora 我 [日期] 要辦 [線上活動／直播／講座]，幫我規劃活動前（預熱報名）、活動中（互動）、活動後（回放導購）三階段的內容與貼文。
```

**5. 產品改版重新上架**
```
/hello-nora 我的 [產品／課程] 改版了，幫我規劃一波「重新上架」宣傳：講清楚升級了什麼、為什麼現在值得買，舊客戶和新客戶各給一套說法。
```

## 🔁 內容生產力

**6. 一篇文章榨成一週素材**
```
/hello-nora 這篇文章幫我變成一週的內容：拆成 3 篇貼文、1 組 IG 輪播、2 支短影音腳本、1 封電子報，語氣維持我的品牌調性。文章：[貼上]
```

**7. 一場直播／影片變一週內容**
```
/hello-nora 這支影片／直播的重點是 [貼上逐字稿或重點]，幫我切成一週可發的素材：金句貼文、重點輪播、短影音腳本各幾則。
```

**8. 舊貼文翻新再放大**
```
/hello-nora 這是我以前成效不錯的貼文：[貼上]。幫我分析它為什麼會中，再用同一個角度重寫 3 個新版本，發不同平台。
```

**9. 一個月內容日曆一次排好**
```
/hello-nora 幫我排下個月的內容日曆：Iris 先給我這個月該主打的主題與關鍵字，Maya 再排成一個月的貼文，對照台灣節日借勢。我的方向是 [方向]。
```

**10. 把客戶常問問題變成一系列內容**
```
/hello-nora 我的客戶最常問這些問題：[列幾個]。幫我把它們變成一系列內容——每題一篇貼文或短影音，順便消除購買疑慮、推一把。
```

## 📈 獲客與成長

**11. 競品逆向工程**
```
/hello-nora 幫我把競品 [名稱／網址] 拆到骨子裡：Iris 拆他的內容與 SEO、Jack 看他的廣告打法、Maya 看他的貼文風格，最後告訴我我可以從哪裡贏他。
```

**12. 漏斗健檢：找出我在哪一段漏人**
```
/hello-nora 幫我做一次行銷漏斗健檢：從陌生人看到我、到追蹤、到購買，幫我看哪一段最可能在漏人、該補什麼內容或動作。我目前的狀況是 [簡述]。
```

**13. 讓 AI 搜尋引擎也找到你（GEO）**
```
/hello-nora 請 Iris 幫我把這篇核心內容優化成「連 ChatGPT、Perplexity、Google AI 都會引用」的版本，並告訴我還該補哪些內容。內容：[貼上]
```

**14. 找出最有機會的關鍵字戰場 + 攻略**
```
/hello-nora 請 Iris 幫我找出 [產品／領域] 最有機會切入的關鍵字戰場，挑 3 個最值得打的，各給我一篇內容攻略（要寫什麼、怎麼贏現在排前面的）。
```

**15. 為冷受眾設計一條「從不認識到買單」的路徑**
```
/hello-nora 幫我為完全不認識我的人，設計一條「滑到我 → 開始追 → 願意買」的內容路徑：每一步該給什麼內容、用哪個平台，產品是 [產品]。
```

## 💰 銷售與轉換

**16. 一頁完整銷售頁**
```
/hello-nora 請 Leon 幫我做 [產品] 的完整銷售頁，從痛點、好處、社證、方案到保證和 CTA，文案和結構都要，給我可預覽的版本。
```

**17. 銷售信序列**
```
/hello-nora 幫我寫一組 [3／5] 封的銷售 email 序列，賣 [產品]：從建立信任、放大渴望、處理疑慮到最後催單，每封一個任務。
```

**18. 異議處理腳本（太貴／沒時間／再想想）**
```
/hello-nora 客戶常因為「太貴／沒時間／再想想」不買，幫我各寫一段不硬銷、站在他立場的回應，能用在客服、私訊和銷售頁 FAQ。產品是 [產品]。
```

**19. 接案／合作提案**
```
/hello-nora 我想接 [品牌／客戶] 的 [合作類型] 案子，幫我寫一份提案：我能幫他解決什麼、怎麼做、為什麼找我。我的優勢是 [簡述]。
```

**20. 把見證變成轉換素材**
```
/hello-nora 這是我收到的學員／客戶見證：[貼上]。幫我把它變成可用的轉換素材——社證輪播、貼文、和銷售頁可放的段落。
```

## 📊 數據與決策

**21. 廣告急救：加碼還是收手**
```
/hello-nora 請 Jack 看這組廣告數據，告訴我該加碼、暫停還是換素材，並說明判斷理由：[貼上 CTR／CPM／ROAS／花費等]
```

**22. 每週成長覆盤**
```
/hello-nora 幫我做這週的成長覆盤：彙整貼文成效、廣告數據、做了哪些事，告訴我哪裡有效、哪裡該調整、下週優先做什麼。
```

**23. 預算重分配：錢有花在對的地方嗎**
```
/hello-nora 我每個月有 [預算] 在行銷上，目前花在 [現況]。請 Jack 幫我看這樣分配對不對，怎麼調整更有效率。
```

**24. A/B 測試規劃 + 判讀**
```
/hello-nora 請 Jack 幫我規劃 [廣告／貼文／銷售頁] 的 A/B 測試：該測哪兩個版本、測多久、怎麼判讀結果。
```

**25. 月報／客戶簡報一鍵生成**
```
/hello-nora 幫我把這個月的成果做成一份簡報，給 [自己複盤／老闆／客戶] 看，重點放在數字成效和下個月策略。
```

## ✨ 個人品牌與經營者

**26. 一句話講清楚你是誰**
```
/hello-nora 幫我把「我是誰、幫誰解決什麼問題」濃縮成一句清楚有力的定位句，多給我幾個風格不同的版本挑。我做的是 [簡述]。
```

**27. 建立你的內容支柱**
```
/hello-nora 幫我建立 3–5 個長期內容支柱（我固定會講的主題方向），讓我的帳號有一致的人設和記憶點。我的專業／產品是 [簡述]。
```

**28. IG 帳號改版健檢**
```
/hello-nora 幫我做一次 IG 帳號健檢：個人簡介、精選動態、九宮格視覺方向，哪裡可以改得更吸引人、更像一個品牌。我的帳號定位是 [簡述]。
```

**29. 把專業變成個人品牌貼文系列**
```
/hello-nora 幫我把我的專業變成一系列個人品牌貼文（Threads／IG），讓人慢慢覺得我是這領域可信的人。我擅長 [簡述]，目標客群 [誰]。
```

**30. 一季行銷作戰藍圖**
```
/hello-nora 幫我規劃接下來一季的行銷作戰藍圖：每個月的主軸、要推什麼、各夥伴怎麼分工，給我一張看得懂、跟得上的全局圖。我這季的目標是 [目標]。
```
